# ZR CAR Centro Automotivo 🚗💨

Website oficial da **ZR CAR Centro Automotivo**, com sede na Vila Matias, em Santos - SP. Trata-se de uma plataforma digital interativa e profissional projetada para clientes simularem orçamentos, buscarem informações sobre serviços automotivos estruturados e entrarem em contato direto através do WhatsApp para agendamentos imediatos.

O design do site segue uma estética moderna, limpa e de alta performance, utilizando um tema escuro sofisticado com acentos em âmbar e tipografia legível para passar credibilidade e solidez técnica ao cliente.

---

## 🌟 Recursos Principais

### 1. 📋 Catálogo Dinâmico de Serviços
* Exibição visualmente refinada de todos os serviços prestados pela ZR CAR:
  * **Alinhamento 3D + Balanceamento**
  * **Manutenção de Freios** (pastilhas, discos, fluido)
  * **Diagnóstico e Troca de Suspensão**
  * **Scanner e Elétrica Computadorizada**
  * **Troca de Óleo e Filtros Essenciais**
  * **Carga de Ar Condicionado e Higienização**
  * **Revisão Preventiva** (Viagem/Check-up)
* Detalhamento inteligente "Ver vantagens e itens inclusos" recolhível para otimizar espaço de leitura.
* Filtros de categoria por interesse e botão integrado de inclusão na simulação de orçamento.

### 🧮 2. Simulador Interativo de Orçamentos (Passo-a-Passo)
* Permite ao cliente calcular uma estimativa de preço instantaneamente, de forma automatizada e amigável:
  * **Passo 1: Seleção de Serviços**: O usuário marca um ou mais serviços do catálogo com exibição em tempo real dos custos estimados individuais.
  * **Passo 2: Informações do Veículo**: Campos inteligentes para preenchimento de Placa (Mercosul/Tradicional), Marca e Modelo do carro.
  * **Passo 3: Dados de Contato & Envio**: O usuário informa o nome e o WhatsApp. O sistema cria e abre uma mensagem de WhatsApp pré-formatada e legível direcionada à equipe da ZR CAR com todos os detalhes da solicitação já calculados.

### 🖼️ 3. Galeria Interativa de Fotos Reais
* Seção de portfólio visual com imagens reais e nítidas das intervenções mecânicas, peças e infraestrutura premium da oficina na Vila Matias.
* Filtro instantâneo por abas (**Tudo, Serviços, Diagnóstico, Infraestrutura**) para fácil navegação do usuário.
* Visualizador ampliado (*Lightbox*) imersivo para detalhamento das imagens, com fechamento intuitivo.

### 💬 4. Central de Provas Sociais e Feedbacks (Depoimentos)
* Coleta e amostragem de avaliações reais de clientes satisfeitos (estrelas, comentários, carimbo de verificado e período).
* Formulário modal interativo e responsivo para novos clientes deixarem seus feedbacks diretamente, atualizando a listagem de depoimentos dinamicamente no estado da aplicação.

### 📍 5. Localização, Contato & Horários
* Integração de Mapa Interativo (Google Maps Embed) apontando a exata localização física da oficina.
* Painel claro indicando horários de funcionamento específicos para a semana e sábados, além de botões com atalhos para copiar endereço ao clipboard em um único clique e telefonar diretamente para o telefone fixo ou fixar contato ao WhatsApp.

## 🛠️ Tecnologias e Arquitetura do Frontend

O projeto foi construído utilizando as ferramentas de desenvolvimento de última geração para proporcionar carregamento ultrarrápido, responsividade primorosa e transições suaves:

* **React 18** — Biblioteca para construção de interfaces modularizadas e robustas.
* **TypeScript** — Fornece tipagem estática e segurança na manutenção de código.
* **Vite** — Build tool veloz para empacotamento de ativos estáticos.
* **Tailwind CSS** — Framework de estilização baseado em classes utilitárias para interfaces refinadas e consistentes com o design system.
* **Motion / Framer Motion (`motion/react`)** — Responsável pelas micro-animações fluidas de abertura de modais, troca de abas, filtros na galeria e transições suaves.
* **Lucide React** — Conjunto moderno de ícones de alta fidelidade e consistência visual para as seções.

## 💻 Como Rodar o Projeto localmente

### Pré-requisitos
Certifique-se de possuir o [Node.js](https://nodejs.org/) instalado em sua máquina.

### Passos para Execução:

1. Instalar as dependências do projeto:
   ```bash
   npm install
   ```
2. Iniciar o servidor de desenvolvimento local:
   ```bash
   npm run dev
   ```
3. Gerar build otimizada de produção:
   ```bash
   npm run build
   ```
